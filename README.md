## 🛠️ Como Executar Este Projeto

### Pré-requisitos
```bash
Node.js 16+
npm ou yarn
```

### Instalação e Execução
```bash
# Clone o repositório
git clone https://github.com/purpesy/portfolio.git
cd portfolio

# Instale as dependências
npm install

# Execute o projeto
npm start

# Acesse http://localhost:3000
```
